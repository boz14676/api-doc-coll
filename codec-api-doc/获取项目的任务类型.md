# 获取项目的任务类型
#API文档/上下游关系

**URL** : `/api/v2/production/task_type`

**Auth required** : NO

**Data constraints**

```json
{
    "project_id": "Integer 项目ID",
}
```

## Success Response

**Code** : `200 OK`

**Content example**

```json
[{
    "task_type_id": "Integer 任务类型ID",
    "task_type_name": "String 任务类型名称",
}]
```
