# 新增上下游关系
#API文档/上下游关系

**URL** : `/api/v2/ud_stream/add`

**Auth required** : NO

**Data constraints**

```json
{
    "task_type_id": "Integer 任务类型ID",
    "task_stage_id": "Integer 任务类型ID",
    "type": "ENUM(UPSTREAM, DOWNSTREAM)",
}
```

> `type ` 类型为字符串。UPSTREAM：上游；DOWNSTREAM：下游。  

## Success Response

**Code** : `200 OK`

**Content example**

```json
{
    "error": "Boolean 0/1",
}
```
