# 联动类型至阶段
#API文档/上下游关系

**URL** : `/api/v2/stage_type/linkage/stage`

**Auth required** : NO

**Data constraints**

```json
{
    "task_type_id": "Integer 任务类型ID",
}
```

## Success Response

**Code** : `200 OK`

**Content example**

```json
[
    {"task_stage_id": "Integer 任务阶段ID", "task_stage_name": "String 任务阶段名称"}
]
```
